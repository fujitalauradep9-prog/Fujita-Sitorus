# Fujita-Sitorus
Smart Visit Carangsari adalah aplikasi web untuk mendata dan memantau kunjungan wisatawan di Desa Wisata Carangsari. Aplikasi ini memungkinkan pengumpulan profil wisatawan secara interaktif serta menampilkan ringkasan data dalam bentuk grafik, sehingga membantu pengelola mengambil keputusan berbasis data dengan lebih mudah dan cepat.
